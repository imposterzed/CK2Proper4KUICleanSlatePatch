# Security Policy

Proper 4K UI Project - CleanSlate Patch is a content mod for Crusader
Kings II — it adds game asset files (DDS icons) and contains no
executable code, network access, or user-data handling. The realistic
attack surface is negligible.

If you discover a vulnerability or game-stability issue with security
implications, please report it via GitHub's **Private vulnerability
reporting** feature on this repository's Security tab, rather than
opening a public issue.

For non-security mod bugs, please use the normal
[issue templates](https://github.com/imposterzed/CK2Proper4KUICleanSlatePatch/issues/new/choose).
